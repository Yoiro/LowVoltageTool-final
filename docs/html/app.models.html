app.models.network module¶

Contains the class Network.

class app.models.network.Network(vnl=230.0)¶

Bases: object

A Network is defined here by its Voltage at Slack Node (called ‘Voltage No Load’). It also contains a list of all brackets, and the methods that will be used to manage them. Be aware that some methods written in this class are not even used in the tool, but are there because they were implemented in an older version, and are there ‘just in case’.

add_bracket(bracket)¶
This method will check if the given node to add is already in the network. If not, it will add it.

Otherwise it will do nothing. :param bracket: the bracket to add to the network. :return: self

delete_bracket(nid)¶

This method will delete a bracket from the network. First it checks if the network contains the id, then whether it is a leaf or not. If it is, it will simply remove it from the network. If it isn’t, it will remove it AND link its children to its parent. :param nid: ID of the node to delete :return: self

method that receives a source node and a destination one. class app.models.networkbranch.NetworkBranch(phases=None, r=None, x=None, length=None, Imax=None)¶

Bases: object

A Branch represents an edge of the Network and thus it stores all informations relative to it: the length of the cable, its length, its resistence, etc…

calculate_impedance()¶

Will calculate impedances on the differant phases. impedance must look like: [[11, 12, 13], [21, 22, 23], [31, 32, 33]] Z[i] = (R[i]+jX[i])*length :return: impedance, a 3 by 3 matrix
